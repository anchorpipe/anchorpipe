# ADR Template

Use this template when authoring a new Architecture Decision Record (ADR).

```
# ADR Title

- Status: Proposed
- Date: YYYY-MM-DD
- ADR ID: ####
- Deciders: anchorpipe Core Program (example: rick1330)
- Context:
  - ...
- Decision:
  - ...
- Consequences:
  - ...
- Alternatives considered:
  - ...
- Revisit criteria:
  - ...
- References:
  - ...
```

1. Copy the template above into a new file named `NNNN-short-title.md`.
2. Update status, date, deciders, context, decision, consequences, alternatives, revisit criteria, and references.
3. Link to related ADRs and issues.
4. Submit via PR and update the ADR index after merging.
